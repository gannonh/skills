# Agent Skills

Personal [Agent Skills](https://agentskills.io/) directory. A mix of original skills and modified open source skills for iOS/macOS development, PR workflows, and general engineering tasks.

## What's here

| Skill                                          | Description                                                               |
| ---------------------------------------------- | ------------------------------------------------------------------------- |
| `agent-browser`                                | Browser automation via CLI for AI agents                                  |
| `auditing-accessibility`                       | VoiceOver, Dynamic Type, color contrast, and App Store accessibility prep |
| `auditing-app-store-compliance`                | App Store guideline audit across all 5 Apple categories                   |
| `automating-ios-simulator`                     | Launch apps, interact with UI, capture screenshots in iOS Simulator       |
| `brainstorming-with-explorer-challenger-teams` | Structured ideation with paired explorer/challenger agent teams           |
| `building-claude-code-skills`                  | Create, edit, and convert Claude Code skills                              |
| `canvas-design`                                | Generate visual art and design assets as PNG/PDF                          |
| `collecting-testflight-feedback`               | Pull and consolidate TestFlight beta feedback from App Store Connect      |
| `committing-changes`                           | Conventional commits with explicit file staging                           |
| `converting-commands-to-skills`                | Migrate slash commands to Skills format                                   |
| `gh-address-comments`                          | Address open review comments on a GitHub PR                               |
| `gh-fix-ci`                                    | Debug and fix failing GitHub Actions checks                               |
| `prd`                                          | Generate Product Requirements Documents                                   |
| `pull-requests`                                | Full PR lifecycle: create, review, merge                                  |
| `ralph`                                        | Convert PRDs to Ralph autonomous agent JSON format                        |
| `running-coderabbit-reviews`                   | Run and process CodeRabbit automated reviews                              |
| `skill-creator`                                | Guide for building new skills                                             |
| `writing-ios-unit-tests`                       | Swift Testing, SwiftData, and coverage workflows                          |

## Installation

Copy individual skill directories into your skills path, typically `~/.claude/skills/` or `~/.agents/skills/`.
